## Tables
- **How to** : Use `|` to divide the columns and `-` for the header of each column.


        | Header 1 | Header 2 |
        | -------- | -------- |
        | cell | cell |
        | cell | cell |
        
        or
        
        Header 1 | Header 2
        -------- | --------
        cell | cell
        cell | cell

    *Result :*
    
    Header 1 | Header 2
    -------- | --------
    cell | cell
    cell | cell
 
 <br>

- **Note** : Use `:` to the left, right, or both sides of the `-` to align.

<br>

    | Header 1 | Header 2 | Header 3 |
    | :-- | :--: | --: |
    |left | middle | right |
    
*Result :*

 | Header 1 | Header 2 | Header 3 |
 | :-- | :--: | --: |
 |left | middle | right |
